## Running Locally

Clone the project

```bash
git clone https://github.com/satnaing/my-portfolio.git
```

Go to the project directory

```bash
cd satnaing.dev
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
